Support

Need help with NetDebug Toolkit? We're here to help with troubleshooting, feature questions, and bug reports.

✉ apps@developerinsider.co

What to include in support requests

  • Device model and iOS/tvOS version
  • App version and build number
  • Tool name (Ping, DNS, Whois, etc.)
  • Target used (domain/IP) and expected result
  • Screenshot or output snippet (if possible)

Common notes

  • Only test hosts and networks you are authorized to query.
  • Results can vary by firewall, DNS policy, and route conditions.
  • Whois and Public IP tools may depend on third-party service availability.
